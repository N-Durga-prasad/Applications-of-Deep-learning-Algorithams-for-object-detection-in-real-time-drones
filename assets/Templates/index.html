Object Detection in Real-time Drone Surveillance

Deep learning (DL) has emerged as a powerful tool in the field of remote sensing, particularly for processing images captured by unmanned aerial vehicles (UAVs). While it has yielded significant contributions across various applications, this review aims to provide a comprehensive understanding of the subject. The study encompasses a broad survey of state-of-the-art techniques and their practical applications on object detection in real time drone surveillance.
